# Structured Data: JSON and SQL in Python

This study covers essential techniques for working with structured data, specifically focusing on **JSON** and **SQL** within a Python data science workflow. The material demonstrates how to seamlessly integrate these data formats into **Pandas DataFrames** for analysis.

---

## 📌 Key Learning Points

### 1. Working with JSON
- Load JSON data into Pandas using `pd.read_json()`.
- Read files directly from the local machine.
- Fetch JSON data from web APIs.

### 2. Working with SQL
- Connect Python to a relational database (e.g., MySQL) using `mysql-connector-python`.

### 3. Data Manipulation
- Use `pd.read_sql_query()` to execute SQL queries directly from Python.
- Filter and extract data into DataFrames for analysis.

### 4. Environment Setup
- Learn the basics of setting up a local database server.
- Use tools like **XAMPP** to manage and interact with structured data.

---

## ✅ Summary
By mastering JSON and SQL integration with Pandas, you can efficiently handle structured data, perform advanced queries, and streamline your Python data science workflow.
