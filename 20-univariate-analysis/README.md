# Exploratory Data Analysis (EDA)

Exploratory Data Analysis (EDA) is the preliminary analysis of data to discover relationships, trends, patterns, and insights using statistics and visualization tools.

## Univariate Analysis

Univariate Analysis is a fundamental part of EDA that focuses on analyzing a single variable at a time, summarizing its data and exploring underlying patterns without considering relationships with other variables.

## Types of Data

Data is categorized into two main types, which dictate the analytical approach:

- **Numerical (Quantitative) Data**: e.g., height, age  
- **Categorical (Qualitative) Data**: e.g., gender, nationality  

---

## Analyzing Categorical Data

- **Countplot**  
  Visualizes the frequency distribution of categorical data, showing how many times each category appears.

- **Pie Chart**  
  Represents the percentage distribution of categories, helping to understand the relative proportion of each category within the whole.

---

## Analyzing Numerical Data

- **Histogram**  
  Displays the frequency distribution of continuous numerical data by grouping values into ranges (bins).

- **Distribution Plot (Distplot / KDE)**  
  Shows the data distribution along with a Kernel Density Estimation curve, highlighting the probability density of different values and determining if the data is skewed.

- **Boxplot**  
  Provides a five-number summary:
  - Minimum  
  - First Quartile (Q1)  
  - Median  
  - Third Quartile (Q3)  
  - Maximum  

  Helps identify the data range, central tendency, and potential outliers.
