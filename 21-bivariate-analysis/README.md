# Exploratory Data Analysis (EDA)

Exploratory Data Analysis (EDA) focuses on understanding relationships between two or more variables to uncover patterns.

The analysis approach depends on the data types of the variables involved:
- Numerical–Numerical  
- Numerical–Categorical  
- Categorical–Categorical  

---

## Key Techniques and Visualizations

### 🔹 Scatter Plots
Used for **numerical–numerical** relationships.  
Example: Analyzing the correlation between total bill amount and tip amount.

---

### 🔹 Bar Plots and Box Plots
Used for **numerical–categorical** analysis.  
Helps compare numerical distributions across categories.  
Example: Comparing average passenger age or fare by class.

---

### 🔹 Distribution Plots (DistPlot)
Used for **numerical–categorical** data.  
Visualizes probability density and distribution differences.  
Example: Comparing age distribution based on survival.

---

### 🔹 HeatMaps and ClusterMaps
Used for **categorical–categorical** analysis.  
Displays relationships in a matrix form using colors and clustering.

---

### 🔹 PairPlots
Used for **multivariate analysis**.  
Visualizes relationships between all pairs of numerical variables.

---

### 🔹 LinePlots
Used for **numerical–numerical (time-based)** data.  
Helps analyze trends over time.  
Example: Passenger growth over years.

---

## 📊 Multivariate Analysis

Multivariate analysis involves analyzing multiple variables together to gain deeper insights and understand complex relationships in the data.
